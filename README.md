# 大学生新生报到智能助手

一个为大学新生提供报到流程指导、信息填写和智能问答的Web应用。

## 功能特性

### 1. 报到流程指南
- 展示报到的4个主要步骤：准备材料、线上注册、现场报到、领取物品
- 每个步骤提供详细说明

### 2. 信息填写
- 收集学生基本信息：姓名、身份证号、专业、联系电话、家庭地址
- 表单验证和提交功能

### 3. 智能问答
- 基于关键词的简单问答系统
- 支持常见问题的自动回复
- 友好的聊天界面

### 4. 常见问题（FAQ）
- 可折叠的问答列表
- 涵盖报到材料、时间、宿舍分配等常见问题

## 技术实现

- **前端技术**：HTML5, CSS3, JavaScript
- **响应式设计**：适配不同设备屏幕
- **用户界面**：现代化的卡片式布局，渐变色彩
- **交互体验**：平滑的动画效果，即时反馈

## 使用方法

1. 打开 `index.html` 文件即可访问应用
2. 浏览报到流程指南了解报到步骤
3. 填写个人信息并提交
4. 使用智能问答功能获取帮助
5. 查看常见问题了解更多信息
